Proyectos

Sistema Gestion de Califiaciones (SGC)

Facilita el control de estudiantes, cursos y calificaciones en instituciones educativas. Utiliza Java (Swing) para la interfaz, MySQL para la base de datos, y JasperReports para generar reportes. Soluciona la automatización del registro de notas, el control de acceso según roles (docentes y administradores) y la generación de informes de rendimiento académico.

Sistema de Gestión de Auto Lavado (CarWash)

facilita la administración de clientes, citas y servicios de un autolavado. Desarrollado en Java (Swing) y MySQL, automatiza la reserva de citas y optimiza la gestión de servicios, mejorando la organización y experiencia del cliente.
